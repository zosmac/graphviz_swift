Graphviz - Graph Visualization Software

GraphvizSwift Help

Graph visualization is a way of representing structural information as diagrams of abstract graphs and networks. It has important applications in networking, bioinformatics, software engineering, database and web design, machine learning, and in visual interfaces for other technical domains. More information on Graphviz is available here - About Graphviz

The GraphvizSwift.app displays graphs written in the Graphviz dot language, using Quartz to render images into several formats, including JPEG, PDF, and SVG. The app also provides a basic text editor for updating the dot language file. The app can also change global attribute settings for the graph and its nodes and edges, enabling one to visualize how changes influence the appearance of the graph. With the text editor one may alter the attributes of particular nodes or edges.

To set attributes, tap the Attributes button at the top right of the graph window. This will open a sidebar that lists attributes and their current global values. Initially, the sidebar presents buttons for selection of graph, node, or edge attributes. Tapping on the appropriate button presents the corresponding attributes.

Tapping on an attribute's name scrolls the attributes documentation window to a brief description of its role. To change an attribute, tap on its value. For text attributes, edit the value in place and press return. For list attributes, select a value from the pop-up menu. A change updates the graph, except for attributes already set in the dot language file. For those, edit the file itself to modify the attributes. More information about Graphviz attributes is available here - Node, Edge and Graph Attributes

Changing the layout

By default, the GraphvizSwift.app uses the dot layout. To change to a different layout, go to the list of graph attributes and look for the layout attribute. Change it as you would any other attribute. For example, to use the neato layout, tap on the layout attribute value, enter neato, and press return.
